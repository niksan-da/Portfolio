Общее описание проекта и исходные данные¶

Стартап "Продукты питания" продаёт продукты питания.

Дизайнеры предложили поменять шрифты во всём приложении. Решение об изменении шрифтов будет принято по результатам A/A/B-теста.

Пользователей разбили на 3 группы: 2 контрольные со старыми шрифтами и одну экспериментальную — с новыми.

**Задача исследования:**

изучить воронку событий и проверить гипотезу об изменении в воронке после внедрения новых шрифтов.

H_0: Статистически значимых различий между долями пользователей, совершивших событие, нет.

H_a: Статистически значимые различия между долями пользователей, совершивших событие, есть.

Пороговое значение alpha = 0.05. Это приемлемый уровень достоверности.

Здесь и далее формулировки гипотез не меняются.

def share_test(event, x, y):
    alpha = 0.05
    users_in_event = np.array([logs_clean_funnel.query('exp_id == @x').\
                               groupby('event').agg(events=('user_id', 'count'), users=('user_id', 'nunique')).\
                               loc[event, 'users'],
                               logs_clean_funnel.query('exp_id == @y').\
                               groupby('event').agg(events=('user_id', 'count'), users=('user_id', 'nunique')).\
                               loc[event, 'users']])
    users_in_group = np.array([logs_clean_funnel.groupby('exp_id').agg(users=('user_id', 'nunique')).\
                               loc[x, 'users'],\
                               logs_clean_funnel.groupby('exp_id').agg(users=('user_id', 'nunique')).\
                               loc[y, 'users']])
    p1 = users_in_event[0]/users_in_group[0]
    p2 = users_in_event[1]/users_in_group[1]
    p_combined = (users_in_event[0] + users_in_event[1]) / (users_in_group[0] + users_in_group[1])
    difference = p1 - p2
    z_value = difference / mth.sqrt(p_combined * (1 - p_combined) * (1/users_in_group[0] + 1/users_in_group[1]))
    distr = st.norm(0, 1) 
    p_value = (1 - distr.cdf(abs(z_value))) * 2
    if p_value < alpha:
        print('Отвергаем нулевую гипотезу: между долями есть значимая разница.')
    else:
        print('Статистически значимых различий между долями пользователей, совершивших событие, нет.')
    
    return print('Событие: {0}, группы: {1} и {2}. p-значение: {3}'.format(event, x, y, p_value))
